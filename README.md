# AI-Powered SEO Content Production — Research Project 🔍🚀

## What This Project Is About
This is a research project on AI-powered SEO content 
production, collecting insights from 10 global experts 
who actively practice and teach this topic.

## Why I Chose This Topic
AI-powered SEO content production is one of the fastest 
growing skills in marketing today. Companies are actively 
hiring people who understand both AI tools and SEO strategy. 
This research helps build a foundation to understand how 
leading practitioners are using AI to produce, optimize 
and distribute content at scale.

## How I Collected the Data
- LinkedIn posts: Manually collected from each expert's 
  public profile (LinkedIn does not allow free API access)
- YouTube transcripts: Automatically fetched using 
  youtube-transcript-api Python library
- Sources: Verified by visiting each profile directly

## Why I Chose These 10 Experts
These experts were selected based on:
- Active posting on LinkedIn (last 30 days)
- Real practitioners, not just commentators
- Blend of SEO expertise AND AI understanding
- Mix of global voices including an Indian practitioner
- High follower counts indicating strong community trust

## Repository Structure
- /research/sources.md — List of all 10 experts with 
  LinkedIn links, follower counts and annotations
- /research/linkedin-posts/ — Recent posts from each 
  expert organized by author
- /research/youtube-transcripts/ — Video transcripts 
  fetched via API organized by expert
- /research/other/ — Additional reference materials

## Tools Used
- Python + youtube-transcript-api — for fetching 
  YouTube transcripts automatically
- Git + GitHub — for version control and publishing
- Claude AI — for research assistance and organization
- Manual LinkedIn research — for collecting posts
